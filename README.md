
# Places Visited Tracker

This is a simple website to keep track of places you have visited. You can add new places with a name, date, and optional notes. All data is stored in your browser (localStorage), so it persists between visits.

## Features
- Add a place with name, date, and notes
- View a list of all places you have added
- Delete places from your list
- Clean, modern design

## Usage
1. Open `index.html` in your web browser.
2. Use the form to add a new place.
3. Your list of places will appear below the form.
4. Data is saved automatically in your browser.

## Files
- `index.html` — Main webpage
- `js. script` — JavaScript logic for adding, displaying, and deleting places
- `style.css` — Styles for the website
